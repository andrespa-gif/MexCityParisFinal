# Timeline Images

Put your image files in this folder. Recommended filenames:

- tenochtitlan.jpg
- colonial.jpg
- reforma.jpg
- porfiriato.jpg
- revolution.jpg
- colonias-populares.jpg
- metro.jpg
- earthquake.jpg
- condesa.jpg
- cablebus.jpg

Then reference them in `index.html` like:

```html
<figure>
  <img src="images/reforma.jpg" alt="Paseo de la Reforma">
  <figcaption>Your caption here.</figcaption>
</figure>
```

Keep filenames simple: lowercase, no spaces, use hyphens.
